Alphabetical Game

Alphabet RL Game is an interactive voice-based learning game where players say words starting with randomly chosen letters.
It uses reinforcement learning logic to adapt difficulty based on user responses, making it fun and educational

DRAG AND DROP

Alphabet RL Game is an engaging drag-and-drop learning game where users match letters with spoken words. It uses reinforcement logic to adapt the letter difficulty based on user performance, making learning interactive and personalized.

Guessing Game

RL Guess Game is an interactive image-based word guessing game built with Flask. Players view pictures, type or speak their guesses, and receive instant feedback with rewards tracked for each attempt.
